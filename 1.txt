<br> 

줄 바꾸기 . 두번쓰면 단락처럼 보일 수 있음 .

​

<p>

paragraph( 단락 ) 의 앞글자를 따온것으로 단락을 나누는 태그이다. 

하지만 일정한 픽셀만큼만 단락을 나누는 역할만 할 수 있기 때문에 <br> 태그보다 자유도가 떨어진다.

​

이것을 보완하기 위해 나온것이 CSS 언어이다. 

예) <p style = "margin-top: 40px;"> ____ </p>

: ____ 글자 위를 40 픽셀만큼 띄운다.

예) <p style = "margin-bottom:40px;"> ____ </p>

: ____ 글자 아래를 40 픽셀만큼 띄운다.

<li> ____ </li> 

목차를 만들 때 사용하는 태그. list 의 앞 두글자를 따왔다. 

부모태그는 <ul> ____ </ul> 이 있다. 

부모태그를 사용하는 이유는 어디까지가 자녀 태그의 영역인지 구분하기 위함이다.

​

예)

<ul>

           <li> ____ </li>

           <li> ____ </li>

           <li> ____ </li>

</ul>

<ul>

           <li> ____ </li>

           <li> ____ </li>

           <li> ____ </li>

</ul>

​

위와 같이 자녀 태그를 구분하기 위해서 <ul> 이라는 부모 태그를 사용한다.

​

<ol> 

이 태그도 <li> 의 부모태그이지만 <ul> 태그와 차이점이 존재한다.

<ul> : unordered list 의 약어로 앞에 숫자가 붙지 않는다.

<ol> : ordered list 의 약어로 <ol> 태그 안에 속하는 자녀 태그들에 한해서 앞에 자동적으로 숫자를 표기한다.

          만약에 <li> 태그를 사용해서 앞에 숫자를 붙여야 할 때, <ol> 태그로 감싸주기만 하면 된다.

Internet VS Web

​

internet 이 web 보다 더 큰 범주이다. 1960 년 internet 이 등장하였고 1990 년 web 이 등장하였다.

info.cern.ch 주소에 접속해 보면 웹의 초기 시작 역사를 볼 수 있다.

​

Client & Server 통신 방식


1. 사용자가 Web Browser 를 통해 http://info.cern.ch/index.html 을 요청함

2. 요청을 받은 web server 는 자신의 하드디스크에서 index.html 을 찾음

3. 그 다음 web server 가 index.html 파일을 전기신호로 바꿔 web browser 에게 줌

4. 응답을 받은 web browser 는 이 전기 신호를 사람이 볼 수 있는 형태로 바꿔서 보여줌

​

Web Server & Web Hosting

​

인터넷이 연결되어 있는 컴퓨터 각각을 host 라고 부름

host 를 빌려주는 회사( server 용 - 항시 켜져 있어야하기 때문 ) 를 웹 호스팅이라고 부름

​

web server 설치 -> apache 설치하기

​

http( hyper text transfer protocol )는 web browser 가 web server 에서 http프로토콜로 정보를 가져오는 것

file:// 로 시작하는 것에는 web server 가 개입하지 않는다. web browser 가 파일안에서 직접 가져오는 것이다.

실습 할 때에는 web server 와 web browser 가 하나의 컴퓨터에 있어서 http:// 와 file:// 의 차이가 없지만 서로 다른 두 컴퓨터가 통신하려면 무조건 http:// 가 필요하다.

​

친구에게 전화하려면 전화번호가 필요하듯이, web browser 가 web server 와 통신하려면 web server 의 

IP 주소가 필요하다. 127.0.0.1 은 전세계 모든 컴퓨터가 자기 자신을 가리키는 IP 주소로 약속한 것이다.
